# Nurse FUKUGYO Lab

看護師の転職・副業メディア「Nurse FUKUGYO Lab」の静的サイトです。

## 公開方法

GitHub Pages の公開元をリポジトリのルートに設定すると、そのまま公開できます。

- 独自ドメイン: `nurse-fukugyoulabo.com`
- ドメイン設定ファイル: `CNAME`
- トップページ: `index.html`

## 構成

```text
.
├── index.html
├── CNAME
├── robots.txt
├── sitemap.xml
└── assets
    ├── css/styles.css
    ├── js/main.js
    └── images/hero-nurse-career.png
```

## 運用メモ

- `SERVICES` のカードリンクは、アフィリエイト審査後に各リンクへ差し替えてください。
- `CONTACT` のフォーム送信先は現在 `mailto:info@nurse-fukugyoulabo.com` です。メールアドレス開設後、またはFormspreeなどのフォームサービス導入後に差し替えてください。
- ブログ記事を増やす場合は、記事ページを追加して `sitemap.xml` にURLを追記してください。
